# Hotwired ATS

This is just my version of the code written in the excellent tutorial [Hotwiring Rails](https://book.hotwiringrails.com)

## Differences

* Uses `vite_ruby`
* Uses docker

## System dependencies

* Docker

## Getting Started

* Build everythign using:

  `script/setup`

* Get it running using:

  `script/restart`

* Get a console using:

  `script/console`
